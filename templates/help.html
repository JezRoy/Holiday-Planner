How to use this Web App

This program is a simple easy holiday/event planner that allows users to manage different tasks that go into making such events work. In this program, anything to-which tasks are being assigned to is called “Event”. If a user of this program chooses to create or open an event, they become an Event Organiser.

Events have 3 phases:

1.Pre-Event – The time before the Event happens and everything leading up to it.

2.During-Event – The time when the event is happening. For example, this may be the user in their holiday destination doing the things they desired to do there.

3.Post-Event – The time after the event, and getting back to normal life.

  

When an Organiser (the user of this program) chooses to create a new event (using the create event button), they will be lead to fill in several fields of information which include their proposed dates for the 3 phases mentioned above, the Event’s name, and its location. The program will automatically calculate the time space between each phase. The program will ask for the organiser’s email and password. This is so that a organiser can send email notifications to “members” (everyone else who is involved in this Event) about when they have tasks assigned to them. It is good idea to save an immediately once it has been created using the save event button.

When logging in or registering for the first time an event can be opened in the event page to be loaded into the task list.

After a program is created or opened, they will be brought to the main planning page which is called the “Task List”. This Task List is a table which will hold all the tasks an organiser wishes to assign to an Event. The Task List also displays the current phase the Event is in, how long until the next phase and the location of the Event. It also holds the button for the notepad function.

A task has 5 main features:

  • Task Name
  • The Member Responsible for the Task
  • The Due Date (and Time remaining)
  • Extra Details (optional)
  • (There is also the option to add a member’s email should the organiser decide to use the email function.)
  • The Progress of the task.

The main 4 features can be seen on the Task List page itself. There is an option to create more tasks, as well as an option to edit a task and delete a task. These tasks can be anything the organiser wishes as long as it can fulfil all the entries mentioned. The user can fill in these fields in a separate window called the Task Creator window.

In addition to this, the notepad function can be used a general notes area and works as an "additional details" space for the entire event. It can be accessed using the notepad button in the navigation bar. To go back to the Task List at any time, click the "Holiday Planner!" button.
